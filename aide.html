Comment modifier un Email ou une Landing Page

✍ Interface de modification visuelle

  • ✏️ Double-cliquez sur un texte pour le modifier.
  • ➕ Utilisez le panneau de droite pour changer la taille, la couleur ou la mise en page.
  • 👇 Cliquez sur un bouton ou une section pour voir ses options de personnalisation.
  • 💾 Cliquez sur "Enregistrer" pour sauvegarder vos modifications.

📧 Modifier un Email

Depuis la page "Mes Emails", cliquez sur le bouton "Modifier" pour ouvrir votre email dans l'éditeur. Une fois les modifications effectuées :

  • Le contenu est automatiquement mis à jour dans votre interface.
  • Si vous souhaitez reprogrammer l'envoi, utilisez la fonction "Programmer un email" dans l'onglet Email.

🌐 Modifier une Landing Page

Depuis la page "Mes Tunnels", une option similaire sera bientôt disponible pour modifier les pages générées par GPT.

⚠️ Important : Vous ne devez pas supprimer les blocs de formulaire de capture ou les balises principales sous peine de dysfonctionnement.
